Introduction to Website Development

HTML

HTML stands for hyper-text markup language. It is used to structure a webpage

CSS

CSS stands for cascading style sheet. It is used to style HTML elements and describe how the webpage should look like.

Javascript

Javascript is a programming language. It is used to program the webpage to become dynamic

Django

Django is a web framework for creating dynamic web pages.
